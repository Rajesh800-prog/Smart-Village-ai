# Smart Village AI 🌾

Smart Village AI is an AI-powered agriculture platform that helps farmers with:

🌦 Weather prediction
🌱 Crop recommendation
🦠 Plant disease detection
📈 Market price insights
🛒 Farmer marketplace

## Tech Stack
Frontend: React + Vite
Backend: Node.js / Python
AI: TensorFlow / Scikit-learn
Deployment: Vercel

## Features
- Farmer dashboard
- Weather alerts
- AI crop recommendations
- Plant disease detection
- Crop marketplace

## Future Improvements
- Voice assistant for farmers
- Mobile app
- IoT farm sensors
